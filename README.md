Group 21 Documentation: https://docs.google.com/document/d/1v2ZnNEi4OqX4airqPOmTqdIYanzd0uZt1qmrwbIpIDY/edit?tab=t.0

commands to run the python project:
1. cd into python-lhdiff-project folder
2. python lhdiff.py Datasets/Aakanksha/File01_Old.py Datasets/Aakanksha/File01_New.py

commands to run python test cases:
1. cd into python-lhdiff-project folder
2. python test_lhdiff.py

commands to run bonus:
1. cd into python-lhdiff-project/bonus folder
2. python bug_detector.py ./gson
3. Open bonus_report.html to see dashboard

To see bonus report for gson repository: https://comp-3110-project.github.io/COMP-3110-Project/python-lhdiff-project/bonus/bonus_report.html

commands to run the java project:
1. cd into java-lhdiff-project folder
2. mvn clean compile
3. mvn exec:java -Dexec.mainClass="LHDiffMain" -Dexec.args="./datasets/Aakanksha/File01_Old.java ./datasets/Aakanksha/File01_New.java"

commands to run java test cases:
1. open java-lhdiff-project folder/src/test/java/LHDiffTest.java
2. run junit tests by right clicking and selecting run test
